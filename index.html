Bem-vindo ao Exemplo de Projeto Git

Sobre este projeto

Este projeto foi criado para demonstrar o uso básico do Git no controle de versão de arquivos de um projeto web. Aqui você pode modificar o conteúdo deste arquivo e versioná-lo utilizando Git.

O que você pode fazer?

  • Criar novas seções nesta página.
    • Alteração nº1
  • Modificar o texto ou o estilo da página.
  • Fazer commit das alterações e enviá-las para um repositório remoto.
  • vamos criar uma nova branch
